Bundled athan recordings — sources and licenses.

Only one bundled recording (doha_fajr.mp3) carries the Fajr-specific
"as-salatu khayrun min an-nawm" line that distinguishes Fajr from
the other four daily calls. The remaining seven are generic adhan
performances; they still play only at Fajr time per the app's
schedule but lack the Fajr-specific phrase.

## Public Domain

All six Public Domain recordings below are free to copy, modify,
and redistribute. No attribution is legally required; the metadata
is preserved here so the origin is never lost when files are
shared outside the app.

  doha_fajr.mp3         Fajr adhan       Public Domain 1.0
  doha_dhuhr.mp3        Dhuhr adhan      Public Domain 1.0
  doha_asr.mp3          Asr adhan        Public Domain 1.0
  doha_maghrib.mp3      Maghrib adhan    Public Domain 1.0
  doha_isha.mp3         Isha adhan       Public Domain 1.0
    Source:  https://archive.org/details/adhan.recordings.from.doha.qatar
    Recorded in Doha, Qatar (2013–2014) and placed in the public
    domain by the uploader via CC Public Domain Mark 1.0.

  sabah_fakhri.mp3      Call to prayer   Public Domain (PD-1923)
    Source:  https://commons.wikimedia.org/wiki/File:Call_to_prayer_by_Sabah_Fakhry.mp3
    Recorded in 1985 by Sabah Fakhri (1933–2021), Syrian classical
    singer and composer. Wikimedia Commons hosts it as PD-1923/
    PD-old based on first-publication date.

Users add more Fajr recordings via the "Add audio file" button in
the Prayer tab. The app plays whichever is selected (or a random
one from the pool) only at Fajr time.
